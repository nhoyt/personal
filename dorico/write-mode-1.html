Dorico Quick Reference

Write Mode – Part 1

Preparing to input notes

Add key and time signatures

Although not strictly required, for newly created scores, especially those created as empty, you should consider specifying both key and time signatures before inputting notes.

Activate note input mode

  • Select a note, rest or bar in the score and press Return or Shift-N.
  • The caret is placed at the beginning of the selection.

Use the following techniques in further preparation of inputting notes and rests.

Position the caret

With note input activated, if you do not want to replace the selected note or rest, use the Right-Arrow key to advance the caret to the desired position.

Alternatively, you can activate insert mode.

Optionally activate chord input

Start chord input in one of the following ways:

  • Press Q, or
  • select the Chords button in the Notes toolbox.

This will allow you to input multiple notes at the same rhythmic position.

Deactivate chord input by pressing Q or selecting the Chords button again.

Optionally activate insert mode

Insert mode allows you to input notes before existing notes without replacing them:

  • Press I to activate insert mode.
  • Input notes and rests as described below.
  • Optionally, press I again to return to normal note input.

To deactivate note input mode

  • Press Esc.

2. Add note by specifying pitch

  • Use the mouse to place the note on the staff at its desired pitch position, or
  • Select a pitch value by pressing one of the letter keys A through G.

You can disable mouse input by activating the ‘Select’ button in the ‘Toolbox’ at the far left, which prevents accidental input of a note when you click in the score.

Adding rests

To input one or more rests while in note input mode, press Space or Right-Arrow. These key commands move the caret, and any gaps between notes are automatically filled with rests.

You can select a different duration by:

  • Pressing the key command of the duration you want, or
  • selecting a note value in the Notes panel.

To insert a rest before a note

  • Move the caret to the note before which you want to insert the rest.
  • Press I to activate insert mode.
  • If ‘Rest’ is not already selected in the Toolbox, press , (comma) or select it in the Toolbox. Note that this command toggles 'Rest' off and on.
  • Optionally select a duration.
  • Press any pitch value A through G (yes, really) to insert a rest of the currently selected duration.

Alternatively, when you shift the rhythmic position of a note to the right, Dorico will automatically fill in the gap with one or more rests corresponding to the distance of the shift.

Selecting notes or items

In order to modify, copy or move notes you have input in your score, you will need to create a selection.

Select one or more notes/items

  • Click a single note/item, and optionally extend the selection (see below), or
  • Shift-Click adjacent notes/items, or
  • Cmd-Click individual notes/items, or
  • make a marquee selection around multiple notes/items.

Move the selection of a single note/item

  • Use the Left-Arrow or Right-Arrow key to move the selection to the previous or next note/item, or
  • use the Up-Arrow or Down-Arrow key to move the selection to a note/item above or below the current position, including across staves.

Extend an existing selection

  • Press Shift-Left-Arrow to extend the selection to the left, or
  • press Shift-Right-Arrow to extend the selection to the right.

Deselect a single note/item

  • Cmd-Click the note/item.

This works for a single note/item selection or within a selection of multiple notes/items.

Deselect all currently selected notes/items

  • Press Cmd-D, or
  • Click outside of the staves within the music area.

To select the first note/rest in your score, using the keyboard:

  • Press Cmd-D to deselect everything in your score.
  • Press an arrow key (e.g., Arrow-Down) to select the first item in the top-left-most music area of the score.

The above method does not seems to work when there is already an item such as a tempo marking at the beginning of the score.
