Dorico Quick Reference

Lead Sheets

Setup Mode

Create and configure project

  • Create an empty project, which places you in Setup Mode. Then add a Solo Player: from Singers select 'Lead'.
  • Change the sound for this player by going into Play Mode and selecting a piano. For example, in Halion, select Yamaha S90ES Piano.
  • In Engrave Mode, change the default font to 'Petaluma': go to Library > Music Fonts.

Write mode

  • In Write Mode, change the clef: press Shift-C and type 'g' for standard G clef.
  • Also change from 'Full Score' to the part named 'Lead' (from the drop-down menu at the top of the window). This is not strictly necessary, but is useful if you begin to change layout parameters.

Layout Options

Launch the Layout Options dialog by pressing Shift-Cmd-L or by selecting it from the Library menu.

Page Setup

  • In the Flows section, set 'Show flow headings' to 'Never'.

Players

  • In the Bar Rests and Multi-bar Rests, set the 'Consolidate' option to 'None'.

Staves and Systems

  • In the Staff Labels section, change both 'Staff labels on first system' and 'Staff labels on subsequent systems' to 'None'.
  • Also in the Staff Labels section, change 'Indent first system of flow by' to '0'.

Notation Options

Launch the Notation Options dialog by pressing Shift-Cmd-N or by selecting it from the Library menu.

Clefs

Under 'Clefs at start of systems following first system', select 'Hide clefs'.

Key Signatures

Under 'Key signatures at start of systems following first system', select 'Hide key signatures'.
