Dorico Quick Reference

Key Commands

Write mode: Insert notation items

Bars or barlines
Shift-B → popover: +3, :|
Clef
Shift-C → popover: treble, bass
Dynamics
Shift-D → popover: pp, mf, ff
Holds & pauses
Shift-H → popover: fer, caesura, comma or , (for breathmark)
Key signature
Shift-K → popover: C, d, Bb, F#, 2f, 3s
Meter/time signature
Shift-M → popover: 3/4, 4/4, 6/8
Chord symbol
Shift-Q → popover: Cmaj7, Bb9, F#min
Repeats
Shift-R → popover: end (for 1st and 2nd endings)
Tempo marking
Shift-T → popover: q=112
Tuplets
; (semicolon) → popover: 3:2, 3:2e (eighth notes)
