Dorico Quick Reference

Preferences

This quick reference is based on the following settings chosen in Dorico > Preferences.

Note: If you add or modify any key commands in Dorico and need to remember which ones, refer to the keycommands.json file in the Application Support/Steinberg/Dorico 5 folder.

Also, it is useful to note that the Help > Key Commands menu item in the Dorico main menu displays all key command settings, including your own customizations.

Note Input and Editing

Pitch and Duration

  • For the setting: By default, specify pitch: choose After Duration.
  • For the setting Specify accidental, rhythm dot and articulations: choose After inputting note.
