Dorico Quick Reference

Preferences

This quick reference is based on the following settings chosen in Dorico > Preferences.

Note: If you add or modify any key commands in Dorico and need to remember which ones, refer to the keycommands.json file in the Application Support/Steinberg/Dorico 5 folder.

Also, it is useful to note that the Help > Key Commands menu item in the Dorico main menu displays all key command settings, including your own customizations.

Note Input and Editing

Pitch and Duration

  • For the setting Specify accidental, rhythm dot and articulations: choose After inputting note.

Note Input > Dotted Notes

The default key command for adding a dot to a note, increasing its duration by half, is '.' — the period character.

To me, it makes sense to add Num'.' — the numeric keypad dot character. However, this overrides the global command 'Move Playhead to Start of Flow'. Consequently, I override the following key command.

Note Input > Set Accidental

Added the following, in addition to their default values:

Natural
Num'0'
Flat
Num'-'
Sharp
Num'='

Play > Move Playhead to > Start of Flow

On a trial basis, I have overridden Num'0' in the global context to execute the command 'Move Playhead to Start of Flow' instead of its default 'Stop Playback'.
